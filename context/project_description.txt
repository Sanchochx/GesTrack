PROYECTO: GesTrack - Sistema de Gestión de Inventario y Pedidos

DESCRIPCIÓN:
Una aplicación web full-stack CRUD que permite a negocios gestionar su inventario de productos, procesar pedidos de clientes, administrar proveedores, y generar reportes de ventas.

STACK TECNOLÓGICO:
- Backend: Python + Flask + PostgreSQL
- Frontend: React (o Vue.js)
- Deployment: Backend en Railway/Render, Frontend en Vercel

ROLES DE USUARIO:
1. Admin (acceso total)
2. Gerente de Almacén (gestión de inventario y proveedores)
3. Personal de Ventas (gestión de pedidos y clientes)

MÓDULOS PRINCIPALES:

1. AUTENTICACIÓN
   - Registro e inicio de sesión
   - Gestión de perfiles de usuario

2. GESTIÓN DE PRODUCTOS
   - CRUD completo de productos (nombre, SKU, descripción, precio de costo, precio de venta, stock, categoría, imágenes)
   - Búsqueda y filtrado de productos
   - Categorías de productos
   - Alertas de stock bajo

3. GESTIÓN DE INVENTARIO
   - Seguimiento de niveles de stock en tiempo real
   - Ajustes manuales de inventario
   - Historial de movimientos de stock
   - Puntos de reorden
   - Valor total del inventario

4. GESTIÓN DE PEDIDOS (ORDERS)
   - Crear pedidos con múltiples productos
   - Cálculo automático (subtotal, impuestos, envío, total)
   - Estados del pedido: Pendiente → Confirmado → Procesando → Enviado → Entregado
   - Estado de pago
   - Historial de pedidos por cliente
   - Procesamiento de devoluciones

5. GESTIÓN DE CLIENTES
   - CRUD de clientes (nombre, email, teléfono, dirección)
   - Historial de compras por cliente
   - Búsqueda de clientes

6. GESTIÓN DE PROVEEDORES
   - CRUD de proveedores
   - Creación de órdenes de compra a proveedores
   - Recepción de mercancía (aumenta inventario)
   - Seguimiento de estado de órdenes de compra

7. REPORTES Y ANALÍTICAS
   - Dashboard con métricas principales (ventas del día, pedidos pendientes, alertas de stock)
   - Reportes de ventas (diario, semanal, mensual)
   - Productos más vendidos
   - Análisis de márgenes de ganancia
   - Reportes de inventario (stock actual, valor del inventario)

FUNCIONALIDADES CLAVE:
- Al crear un pedido, el stock se reduce automáticamente
- Alertas cuando productos están por debajo del punto de reorden
- Cálculo automático de márgenes de ganancia
- Historial completo de movimientos de inventario
- Búsqueda y filtros en todas las secciones
